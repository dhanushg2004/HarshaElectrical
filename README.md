# ⚡ Harsha Electrical

A professional electrical services website built using HTML and CSS.

## 🌐 Live Website

https://harshaelectrical.netlify.app/

## 📌 Project Overview

Harsha Electrical is a modern business website designed for an electrical services and materials supplier. It provides information about services, electrical materials, contact details, and online booking.

## ✨ Features

- Responsive Design
- Professional Home Page
- About Section
- Electrical Services
- Before & After Showcase
- Electrical Materials List
- Booking Form
- Google Maps Location
- WhatsApp Contact Button
- Mobile Friendly

## 🛠️ Technologies Used

- HTML5
- CSS3
- Responsive Web Design

## 📂 Project Structure

```
HarshaElectrical/
│── index.html
│── style.css
│── README.md
```

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/dhanushg2004/HarshaElectrical.git
```

2. Open the project folder.

3. Double-click **index.html** or open it using Live Server in VS Code.
